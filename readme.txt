[製作環境]
言語：HTML, CSS, JavaScript(jQuery)
文字コード：UTF-8
OSXのChromeで動作を確認

[仕様]
投稿はチェックと内容からなる。
投稿データはローカルストレージに保存する。
新しい投稿ほど上に表示される。
HTMLタグが入力された場合はエスケープする。

[独自機能]
◉「全削除」ボタンの追加
デバック用の機能。TODOが全て削除される。

◉TODOリストの先頭部に、今までのTODOの遂行回数を表示
「初期化」ボタンを押すと、0にリセットされる。

◉各TODOに「遂行」ボタン、「編集」ボタン、「削除」ボタン、TODOの追加日時を表示した。
-「遂行」ボタン
TODOが削除され、遂行回数が増える。

-「編集」ボタン
TODOの内容を編集できる。
このボタンを押すと、入力フォームと「変更」ボタンが表示される。
「変更」ボタンを押すと、入力フォームの内容に更新される。

-「削除」ボタン
TODOが削除される。
